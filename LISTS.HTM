Lists in HTML

In HTML, there are three types of lists: unordered lists, ordered lists, and definition lists.

Unordered List

An unordered list is a list of items that have no particular order. Each item in the list is marked with a bullet point. The syntax for creating an unordered list is:

		<ul>
			<li>Item 1</li>
			<li>Item 2</li>
			<li>Item 3</li>
		</ul>

Here is an example of an unordered list:

  • Item 1
  • Item 2
  • Item 3

Ordered List

An ordered list is a list of items that have a specific order. Each item in the list is marked with a number or letter. The syntax for creating an ordered list is:

		<ol>
			<li>Item 1</li>
			<li>Item 2</li>
			<li>Item 3</li>
		</ol>

Here is an example of an ordered list:

  1. Item 1
  2. Item 2
  3. Item 3

Definition List

A definition list is a list of terms and their corresponding definitions. The syntax for creating a definition list is:

		<dl>
			<dt>Term 1</dt>
			<dd>Definition 1</dd>
			<dt>Term 2</dt>
			<dd>Definition 2</dd>
			<dt>Term 3</dt>
